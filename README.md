# Final-Year-Project
This is my Final Year Project implemented using a technology stack that includes Java, J2EE, Spring Boot, Spring Security, Spring Web, Spring Data JPA, Hibernate, Spring Dev Tool, Spring MVC, Spring IOC, Thymeleaf, HTML, CSS, JavaScript, Bootstrap 5, jQuery, MySQL, Apache Tomcat, Eclipse, and VS Code.
**Project Overview:**
The project involves the implementation of a web application with three distinct roles:
1-Student
2-Tutor
3-Admin.
**Technologies Used**
Java: The core programming language.
J2EE: The Java Platform, Enterprise Edition, for building enterprise-level applications.
Spring Boot: A framework for creating stand-alone, production-grade Spring-based applications.
Spring Security: Provides authentication and authorization support.
Spring Web: Enables the development of web applications using the Spring framework.
Spring Data JPA: Simplifies data access using the Java Persistence API (JPA).
Hibernate: An object-relational mapping framework.
Spring Dev Tool: Enhances the development experience with additional tools.
Spring MVC: A model-view-controller framework for building web applications.
Spring IOC: Inversion of Control container for managing JavaBeans.
Thymeleaf: A modern server-side Java template engine.
HTML, CSS, JavaScript: Standard web development technologies.
Bootstrap 5: A popular CSS framework for responsive and mobile-first front-end development.
jQuery: A fast and feature-rich JavaScript library.
MySQL: A relational database management system.
Apache Tomcat: A web server and servlet container.
Eclipse, VS Code: Integrated development environments (IDEs) for coding and development.
